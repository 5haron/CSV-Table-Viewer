## Dependencies used in CSV Table Viewer:

- **`pandas`:** A popular data manipulation library for Python that provides data structures and functions needed to work with structured data seamlessly. It is used in this program to read, sort, filter, and write data in a CSV file into a DataFrame, which is a two-dimensional tabular data structure with labeled axes (rows and columns).

- **`tabulate`:** a Python library for generating human-readable ASCII tables. It is used in this program to print a formatted version of the DataFrame to the console, making it easy for users to visualize the data.
